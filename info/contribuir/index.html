Como contribuir

  1. Faça um fork do repositório do projeto.
  2. Clone o fork na sua máquina:

git clone https://github.com/{seu-usuario}/T2G3-Sistema-Instalacao-Eletrica.git

  1. Comente na issue que deseja contribuir ou crie uma issue nova.
  2. Entre no repositório clonado na sua máquina:

cd T2G3-Sistema-Instalacao-Eletrica

  1. Após enviar suas contribuições para o fork do seu repositório, faça um pull request.
  2. Aguarde a revisão.
